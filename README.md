# 📖 Harpa Cristã App

Aplicativo móvel com os 640 hinos da Harpa Cristã, desenvolvido com React Native (Expo).

## ✨ Funcionalidades

- ✅ Lista completa com 640 hinos
- 🔍 Busca por número ou título
- ⭐ Favoritos (salvos localmente)
- 📱 Totalmente offline

## 🛠️ Tecnologias

- React Native (Expo)
- React Navigation
- AsyncStorage
- JavaScript

## 🚀 Como executar

```bash
git clone https://github.com/eduardo7321/harpa-crista-app.git
cd harpa-crista-app
npm install
npx expo start
```

## 📱 Download

- Escaneie o QR Code com o Expo Go para testar no seu celular.
